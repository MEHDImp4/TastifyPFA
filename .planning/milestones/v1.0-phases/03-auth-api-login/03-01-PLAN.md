<threat_model>
## Trust Boundaries
| Boundary | Description |
|----------|-------------|
| Browser -> API | Refresh token stored in HttpOnly cookie to prevent JS access. Access token in memory. |

## STRIDE Threat Register
| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Information Disclosure | Refresh Token | Mitigate | HttpOnly cookie prevents XSS exfiltration. |
| T-03-02 | Spoofing | Access Token | Mitigate | Short-lived (15m) access tokens. |
| T-03-03 | Tampering | JWT Signature | Mitigate | RS256 or HS256 with strong SECRET_KEY (SimpleJWT default). |
</threat_model>
