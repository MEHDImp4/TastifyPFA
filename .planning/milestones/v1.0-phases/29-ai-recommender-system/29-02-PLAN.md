  <behavior>
    - Test 1: `GET /api/menu/plats/<id>/recommendations/` returns 200 with a list of serialized `Plat` objects.
    - Test 2: If the `id` exists in the `plat_similarities` cache, it returns those specific `Plat` objects.
    - Test 3: If the `id` is not in the cache (cold start) or cache is empty, it returns the top 5 most frequently ordered active Plats as a fallback.
  </behavior>
  <action>
    1. Add an `@action(detail=True, methods=['get'])` named `recommendations` to `PlatViewSet`.
    2. In the action, retrieve `plat_similarities` from `django.core.cache.cache`.
    3. If `self.get_object().id` is in the similarities dict, fetch those `Plat` objects from the DB (ensure they are `active()` and `est_disponible=True`).
    4. If not found in cache, calculate the top 5 most popular `Plats` by annotating with `Count('lignes_commande')` and order by `-lignes_count`.
    5. Serialize the resulting QuerySet with `self.get_serializer(plats, many=True)` and return the Response.
  </action>
  <verify>
