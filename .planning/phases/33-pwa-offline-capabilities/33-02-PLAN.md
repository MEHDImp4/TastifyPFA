# Plan 33-02: Offline Support with Workbox

## Goal
Configure Workbox via `vite-plugin-pwa` to enable asset caching and offline shell.

## Tasks
1.  **Workbox Config (Back-Office)**:
    - [ ] Configure `runtimeCaching` for fonts and static images.
    - [ ] Enable `globPatterns` for precaching the entire shell.
2.  **Workbox Config (Portail)**:
    - [ ] Configure `runtimeCaching` for menu images (maybe Cache-First).
    - [ ] Enable `globPatterns` for precaching.
3.  **Connectivity Components**:
    - [ ] Create `shared/components/ConnectivityBanner.tsx` to show "You are offline".
    - [ ] Integrate banner into both apps.
4.  **Testing Offline Mode**:
    - [ ] Simulate "Offline" in DevTools.
    - [ ] Verify app shell loads.

## Verification
- [ ] Offline banner appears when network is cut.
- [ ] App remains interactive (UI transitions work) while offline.
- [ ] Refreshing the page while offline still loads the app shell.
