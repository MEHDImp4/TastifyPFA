# Plan 33-01: PWA Setup & Manifest

Install `vite-plugin-pwa` and configure the web app manifest for both frontends.

    - [x] `npm add -D vite-plugin-pwa` in `app/frontend/backoffice`.
    - [x] `npm add -D vite-plugin-pwa` in `app/frontend/portail`.
2.  **Icons Generation**:
    - [x] Create `public/icon-192x192.png` and `public/icon-512x512.png` (or SVGs) in both apps.
3.  **Vite Configuration (Back-Office)**:

4.  **Vite Configuration (Portail)**:
