<objective>
Automate the daily generation of restaurant checklists using Celery Beat, ensuring that every operational day begins with a fresh set of procedures.

Purpose: Eliminate manual reset of checklists and ensure accountability.
Output: Periodic Celery task scheduled for daily execution.
</objective>

<feature>
  <name>Daily Checklist Generation Task</name>
  <files>app/backend/apps/checklists/tasks.py, app/backend/apps/checklists/tests/test_tasks.py</files>
  <behavior>
    - Task: `generate_daily_checklists`
    - Behavior: 
        1. Fetch all `Checklist` templates where `active=True`.
        2. For each template, check if a `ChecklistExecution` already exists for today's date.
        3. If not, create a `ChecklistExecution` and generate empty `ChecklistItemResponse` entries for every `TaskChecklist` in that template.
  </behavior>
  <implementation>
    - Use `@shared_task` decorator.
    - Use `transaction.atomic` for mass creation of responses.
    - Log results (number of checklists generated).
  </implementation>
</feature>
