# Plan 09-02: Map Editor (GERANT only)

## Objective
Enable Gérants to arrange tables on the floor plan via drag-and-drop. This will persist `pos_x` and `pos_y` to the backend.

## Tasks

### 1. Feature: Edit Mode Toggle
- Add "Modifier le plan" button in `MapView.tsx`, visible only if `user.role === 'GERANT'`.
- State: `isEditMode`.

### 2. Component: Draggable TableItem
- If `isEditMode` is true, enable dragging using `framer-motion` `drag` prop.
- Constraint: Dragging should be limited to the `TableMap` SVG container.
- Visuals: Add a dotted border or "grab" cursor when in edit mode.

### 3. Persistence: Save Positions
- Add "Enregistrer" and "Annuler" buttons when in edit mode.
- Logic: Collect all new positions and send `PATCH /api/tables/{id}/` for each changed table.
- Optimistic UI: Update local state during drag, but revert if save fails.

## Success Criteria
- [ ] Gérants see an "Edit" button.
- [ ] Tables can be moved freely on the map in Edit Mode.
- [ ] Positions are persisted to the database and survive a refresh.
