## 3. Feature: GERANT Edit Mode
type: implementation
files:
  - `frontend/salle/src/pages/Map/MapView.tsx`
  - `frontend/salle/src/components/map/TableMap.tsx`
  - `frontend/salle/src/components/map/TableItem.tsx`
  - `frontend/salle/src/pages/Map/MapView.test.tsx`

  - Show edit controls only when `user.role === 'GERANT'`.
  - Add `isEditMode`, local table position state, and `dirtyTables` tracking in `MapView.tsx`.

acceptance_criteria:
  - Non-GERANT users cannot access editor behavior through visible UI.
  - Entering and leaving edit mode does not lose the last fetched table state.

## 4. Interaction: Drag, Snap, and Collision Feedback
type: implementation
files:
  - `frontend/salle/src/components/map/TableMap.tsx`
  - `frontend/salle/src/components/map/TableItem.tsx`
  - `frontend/salle/src/components/map/TableMap.test.tsx`
action:
  - Enable drag only in edit mode.
  - Convert drag-end movement into the SVG coordinate system before updating local state.
  - Snap updated positions to the nearest 20px increment and clamp them inside the `1000x800` viewBox.
  - Detect simple bounding-box overlaps and render a red glow warning on overlapping tables.
  - Treat collision feedback as advisory only; do not block save.
verify:
  - Tests cover snap math, clamping, and overlap detection.
acceptance_criteria:
  - Dragged positions always land on 20px grid increments.
  - Overlaps are visible before saving.
  - Touch and pointer interactions use the same edit-mode guard.

## 5. Persistence: Batch Save and Cancel
type: implementation
files:
  - `frontend/salle/src/pages/Map/MapView.tsx`
  - `frontend/salle/src/pages/Map/MapView.test.tsx`
action:
  - Add "Enregistrer" and "Annuler" actions while edit mode is active.
  - Save only dirty tables with individual `PATCH /api/tables/{id}/` requests containing `pos_x` and `pos_y`.
  - Keep the UI in edit mode and show an error if any PATCH request fails.
  - On successful save, clear dirty state, exit edit mode, and refresh from `GET /api/tables/`.
  - On cancel, restore the last fetched table collection and clear dirty state.
verify:
  - Tests assert only dirty tables are patched.
  - Tests assert cancel restores the last fetched positions.
  - Tests assert failed saves keep unsaved local positions visible.
acceptance_criteria:
  - Saved positions survive refresh because they are persisted through the Phase 8 API.
  - Partial save failures are surfaced without silently discarding local edits.
