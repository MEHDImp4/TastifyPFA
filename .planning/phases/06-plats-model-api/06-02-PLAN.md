    ```python
    class PlatViewSet(viewsets.ModelViewSet):
        serializer_class = PlatSerializer

        def get_permissions(self):
            if self.action in ('list', 'retrieve'):
                return [IsAuthenticated()]
            return [IsAuthenticated(), IsGerant()]

        def get_queryset(self):
            user = self.request.user
            if user.is_authenticated and user.role == 'GERANT':
                qs = Plat.objects.all()
            else:
                qs = Plat.objects.active().filter(est_disponible=True)

            if categorie_id:
                qs = qs.filter(categorie_id=categorie_id)
            return qs.order_by('categorie', 'nom')

        def destroy(self, request, *args, **kwargs):
            instance = self.get_object()
            instance.delete()
            return Response(status=status.HTTP_204_NO_CONTENT)
    ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → POST /api/plats/ | Untrusted multipart data (nom, prix, image) enters here |
| client → DELETE /api/plats/{id}/ | Must not be callable by non-GERANT |
| get_queryset → DB | Non-GERANT must never receive inactive OR unavailable dish rows |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-04 | Spoofing | `PlatViewSet.get_permissions()` | mitigate | `IsGerant` enforced on create/update/destroy. Non-GERANT → HTTP 403. Unauthenticated → HTTP 401. |
| T-06-05 | Information Disclosure | `PlatViewSet.get_queryset()` | mitigate | Non-GERANT sees only `active() AND est_disponible=True`. Inactive/unavailable dishes return 404 on direct GET by ID — filtered at ORM level before serialization. |
| T-06-06 | Tampering | `destroy()` | mitigate | Calls `instance.delete()` (model-layer soft delete). `super().destroy()` is never called — hard delete is blocked. |
| T-06-07 | Tampering | `ImageField` multipart | mitigate | `ImageField(use_url=True)` with Pillow validates MIME type before saving. `upload_to='plats/'` restricts storage to a controlled path. |
</threat_model>
