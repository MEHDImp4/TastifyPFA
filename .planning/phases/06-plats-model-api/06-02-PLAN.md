    Implementation Details:
    - Inherit from viewsets.ModelViewSet.
    - Set serializer_class = PlatSerializer.
    - get_permissions(): 
        - 'list', 'retrieve' actions: IsAuthenticated().

        - Otherwise: return Plat.objects.active().filter(est_disponible=True).

        - Return Response(status=status.HTTP_204_NO_CONTENT).
