# Plan 11-02: Table State Synchronization

## Goal
Automate table status transitions based on order lifecycle using Django signals.

## Proposed Changes

### backend/apps/commandes/signals.py
- Add `post_save` receiver for `Commande`.
- Logic:
  - If `created`: `instance.table.statut = Table.Statut.OCCUPEE`.
  - If `statut` changed to `PAYEE` or `ANNULEE`: `instance.table.statut = Table.Statut.LIBRE`.
  - Use `update_fields=['statut']` on table save.

### backend/apps/commandes/tests/test_table_sync.py
- Test: Creating order sets table to `OCCUPEE`.
- Test: Paying order sets table to `LIBRE`.
- Test: Cancelling order sets table to `LIBRE`.
- Test: Updating order other fields does not change table status.
