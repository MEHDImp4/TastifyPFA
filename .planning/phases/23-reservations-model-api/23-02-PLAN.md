# Phase 23-02 Plan: Reservations API & RBAC

## Goal
Expose the reservations via a REST API with strict role-based access control.

## Tasks
1. **Serializers**: Create `ReservationSerializer` in `apps/reservations/serializers.py`.
2. **ViewSets**: Implement `ReservationViewSet` in `apps/reservations/views.py`.
    - `get_queryset`: Filter by `client` for non-staff.
    - `perform_create`: Set `client=request.user` for client bookings.
3. **Permissions**: Use `IsAuthenticated` and custom logic to ensure clients can't modify status (other than cancelling).
4. **URL Routing**: Register endpoints in `api_router.py`.

    - `test_overlap_returns_400`.
