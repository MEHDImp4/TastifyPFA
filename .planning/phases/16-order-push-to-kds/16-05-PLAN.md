```css
}
```

Notes:
- `rgba(42, 157, 143, X)` is the literal RGB of `--color-teal` (#2A9D8F). We use rgba (not the CSS variable) because `box-shadow` cannot consume an HSL CSS variable directly without `color-mix` or `hsl()` substitution; rgba is portable across all targeted browsers.
- The animation runs `infinite` deliberately. The TicketCard component will toggle the class on/off via React state (Plan 06). After 10 seconds the React effect removes the class — the CSS keyframe does not need a fixed iteration count.
- Tailwind v4 will pass through this raw CSS as-is (no @theme block needed because we are not registering a token — we are emitting plain CSS).
