# Phase 30, Plan 03: Frontend Integration

## Goal
Integrate the review submission and viewing into the React SPAs.

## Context
The Portail Client needs a UI to submit reviews for dishes they've ordered. The Back-Office (staff SPA) needs a view for managers to monitor feedback and the AI-determined sentiment.

## Steps
1. **Portail Client**:
   - Add an API client function for `POST /api/avis/`.
   - Add a Review Modal/Section to the client's order history or dish details view, allowing them to submit a text review and a star rating.
   - Handle the success state with a toast notification.
2. **Back-Office**:
   - Add an API client function for `GET /api/avis/`.
   - Create a 'Reviews' tab or dashboard widget in the Gerant view showing recent reviews, the user's rating, and the AI-determined `sentiment_score`.
3. Add tests (unit/integration) for the new frontend components.
